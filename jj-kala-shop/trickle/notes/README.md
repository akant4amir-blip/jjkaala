# جی جی کالا - فروشگاه اینترنتی

## درباره پروژه
فروشگاه اینترنتی جی جی کالا یک سیستم کامل فروشگاهی آنلاین با پنل مدیریت است.

## ساختار پروژه
- `index.html` - صفحه اصلی فروشگاه
- `admin.html` - پنل مدیریت
- `app.js` - لوجیک اصلی برنامه
- `admin-app.js` - لوجیک پنل مدیریت
- `components/` - کامپوننت‌های React

## رمز عبور پنل مدیریت
```
JjKala@2025!Admin#Secure$123
```

## دسترسی به پنل مدیریت
برای ورود به پنل مدیریت به آدرس `admin.html` بروید و رمز عبور بالا را وارد کنید.

## امکانات
- نمایش محصولات
- دسته‌بندی محصولات
- پنل مدیریت با احراز هویت
- افزودن، ویرایش و حذف محصولات
- ذخیره‌سازی در دیتابیس Trickle
